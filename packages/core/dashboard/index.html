🎭 Playwright

Add to your fixtures.ts and call locatorGuard.check() in any test.

import { createGuardFixture }
  from '@dom-locator-guard/playwright';

export const test = createGuardFixture({
  config: { outputFormats: ['html'] }
});

// In your test:
await locatorGuard.check('login-page');

🌲 Cypress

Register the plugin in cypress.config.ts, then use cy.guardSnapshot().

// cypress.config.ts
import { locatorGuardPlugin }
  from '@dom-locator-guard/cypress';

setupNodeEvents(on, config) {
  locatorGuardPlugin(on, config);
}

// In your test:
cy.guardSnapshot('checkout');

🐍 Selenium (Python)

Wrap your driver with LocatorGuard and call guard.check().

from locator_guard import LocatorGuard

guard = LocatorGuard()
driver.get("/login")
report = guard.check(driver, "login")

if report and report["changedCount"] > 0:
  print("⚠ Locator changes detected")

⌨️ CLI

Compare HTML files offline without a running browser.

# Save baseline
locator-guard save-baseline \
  --input ./login.html \
  --feature "Login Page"

# Compare against baseline
locator-guard diff \
  --feature "Login Page" \
  --current ./login-new.html \
  --format console,html

⚙️ GitHub Actions

Auto-detect locator changes on every PR that modifies HTML or templates.

# .github/workflows/locator-guard.yml
on:
  pull_request:
    paths: ['**/*.html']

jobs:
  locator-guard:
    steps:
      - uses: actions/checkout@v4
      - run: npx locator-guard diff \
          --feature login \
          --current ./login.html

🔧 Configuration

Tune thresholds and tracked attributes via config.

const config = {
  // 0-1: how similar to be "same element"
  similarityThreshold: 0.75,

  // attributes to track
  trackedAttributes: [
    'id', 'dataTestId', 'dataCy',
    'ariaLabel', 'role', 'class'
  ],

  // ignore dynamic sections
  ignoreSelectors: ['.ad-unit'],

  outputFormats: ['console', 'html'],
};
